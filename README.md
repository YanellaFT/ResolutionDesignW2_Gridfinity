# ResolutionDesignW2_Gridfinity

### Duck
1x2 boxes (1 = 41.5mm)

Can hold one rubber duck and one pirate duck figurine I received at UCF Programming Competition. 

Instead of extruding like on the other designs, I revolved the shape to make sure the duck fits properly because the back is not flat for these ducks.


### Headphone
7x7 boxes (1 = 41.5mm)

Can hold my A18 headphones. 


### Scissor
3x4 boxes (1 = 41.5mm)

Can hold my favorite pair of scissors.
